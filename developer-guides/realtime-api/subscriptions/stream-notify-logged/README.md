# stream-notify-logged

Stream for logged users

Events available:

- Users:NameChanged
- Users:Deleted
- updateAvatar
- updateEmojiCustom
- deleteEmojiCustom
- roles-change

```json
{
    "msg": "sub",
    "id": "unique-id",
    "name": "stream-notify-logged",
    "params":[
        "event",
        false
    ]
}
```
