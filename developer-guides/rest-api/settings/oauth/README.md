# Settings oAuth

A simple method, requires no authentication, that returns list of all available oauth services.

| URL | Requires Auth | HTTP Method |
| :--- | :--- | :--- | :--- |
| `/api/v1/settings.oauth` | `no` | `GET` |

## Example Call

```bash
curl http://localhost:3000/api/v1/settings.oauth
```
